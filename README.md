# BMI Clinical Analyzer
 This is a simple Python-based BMI calculator designed with a clinical perspective.
## Features
- Calculates Body Mass Index (BMI)
- Provides classification (Underweight,Normal,Overweight,Obese)
- Includes basic clinical risk interpretation
## Purpose
 This project demonstrates the integration pf healthcare knowledge with basic programming skills.
## How to Run
 1. Open the file in Python 
 2. Enter weight in kilograms
 3. Enter height in meters
 4. Get BMI result with clinical interpreatation
